# Backroom Lounge — Osogbo

_Cafe · Culture · Retreat · Community_

Backroom Lounge is a premium, intentional multipurpose space located in the heart of Osogbo, Nigeria. It is designed as a sanctuary for those seeking a blend of cultural discovery, quiet reflection, and curated social experiences. From artisanal refreshments to a dedicated on-site cultural museum, Backroom Lounge is where calm culture meets classy lounging.

---

## Brand Philosophy

At Backroom Lounge, we believe in the power of "The Reset." Our space is built on four core pillars:
- **Cafe & Refreshments:** A curated selection of premium beverages and light fare.
- **Cultural Museum:** A dedicated space for heritage observation and local discovery.
- **Retreat Energy:** An atmosphere engineered for quiet work, reading, and deep thought.
- **Curated Gatherings:** A venue for intentional community events that celebrate heritage and spirit.

---

## Tech Stack & Performance

This digital platform is built with a focus on high-performance, minimalist engineering to mirror the lounge's commitment to quality and elegance.

| Technology | Implementation |
| :--- | :--- |
| **React** | Component-based architecture for seamless navigation. |
| **Vite** | Optimized build pipeline for lightning-fast load times. |
| **Tailwind CSS** | Utility-first styling for a consistent, premium design system. |
| **TypeScript** | Strict typing for reliability and maintainability. |
| **Framer Motion** | Sophisticated, hardware-accelerated animations. |

### Performance Metrics
- **Zero-Bypass Interactions:** Utilizing native browser APIs for buttery-smooth scrolling.
- **Optimized Assets:** High-fidelity imagery with adaptive loading for global accessibility.
- **Mobile-First:** Fully responsive design ensuring a premium experience on any device.

## Development

### Prerequisites
- Node.js (Latest LTS recommended)

### Getting Started
1. **Clone the repository**
   ```bash
   git clone https://github.com/your-org/backroom-lounge.git
   cd backroom-lounge
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Launch Development Server**
   ```bash
   npm run dev
   ```

### Production Build
To generate an optimized production build:
```bash
npm run build
```
